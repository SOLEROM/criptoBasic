## install

* from src 

```
git clone git@github.com:libbitcoin/libbitcoin-explorer.git
mkdir ~/libbitcoin
./install.sh --prefix=/home/XXX/libbitcoin --build-boost --build-zmq --disable-shared


```
* run downloaded elf
